# My Reading Journey

- This is a simple web app made for keeping track of book that you have read. You can add book that you have read with minimum details like 'Title', 'Author', 'Genre', 'Description', 'Date Of Reading Started' and 'Date Of Reading Finished'. Also You can add 'Cover Image' of the book and also 'Rate' the book, out off 5 stars.

- All the other details are not necessary except 'Title'.


## Additional Features

- User Sign Up and Login, i.e., user authentication.
- Displays small book cards on home page and full details when clicked on card.
- Edit and Delete book after adding it.
- Light and Dark theme.
